CRM Application for Laptop Rentals

Description :-
      This project is a Salesforce-based CRM application for managing laptop rentals. It utilizes Salesforce tools to streamline the rental process, track inventory, and automate 
tasks like approvals and reporting.

Purpose :-
The system was developed to provide organizations with an efficient way to manage their laptop inventory and rental requests, reducing manual effort and improving accuracy.

Features :-
1)Efficient laptop inventory management.
2)Automated handling of rental requests and approvals.
3)Administrator dashboard for monitoring and managing rentals.

Technologies Used :-
1)Salesforce CRM (Apex, Visualforce, Lightning Components).
